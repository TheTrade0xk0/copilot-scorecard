How to get an Anthropic API Key

1
Go to console.anthropic.com and sign up or log in.
2
In the left sidebar, click API Keys.
3
Click Create Key, give it a name, and copy the key. It starts with sk-ant-.
4
Add a payment method under Billing. New accounts get $5 free credits — enough for ~125 research runs.
5
Paste the key into the field above. It is never stored — used only for your request.
